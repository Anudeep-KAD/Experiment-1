# 🚀 Multi-Module Web Application

A simple and clean **multi-page web application** that integrates three core modules into one unified interface. This project is built using **HTML, CSS, and JavaScript** and demonstrates fundamental frontend development concepts.

---

## 📖 Overview

This project consists of three independent modules:

* 🧾 **Student Registration System**
* 💰 **Banking Interface (Deposit & Withdraw)**
* 📊 **Admin Dashboard UI**

All modules are connected through a central navigation page (`index.html`) for easy access.

---

## ✨ Features

### 🧾 Student Registration

* Input validation for:

  * Full Name (9–50 characters)
  * Email format
  * Age (16–30)
* Real-time error messages
* Simple and user-friendly UI

---

### 💰 Banking System

* Deposit money into account
* Withdraw money with balance check
* Dynamic balance updates
* Error handling for invalid inputs

---

### 📊 Admin Dashboard

* Modern UI with dark theme
* Navigation sidebar
* Dashboard cards:

  * Analytics
  * Users
  * Settings
